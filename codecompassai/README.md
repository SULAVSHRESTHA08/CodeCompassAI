# CodeCompassAI

CodeCompassAI is a local-first, context-aware developer assistant designed to serve as your extension-based code memory assistant. It helps you resume your previous programming sessions smoothly by tracking your edits and providing AI-generated insights.

## Features

- **Session Context Telemetry**: Monitors saves and file edits to automatically track active files.
- **Git Telemetry Integrator**: Automatically captures git diff and git status in your active workspace workspace.
- **Secure Token Storage**: Safely stores your Gemini API key inside VS Code's SecretStore.
- **Modern UI Summary**: Renders clean Markdown reports with beautiful glassmorphism theme styling matching your current VS Code theme.

## Setup & Configuration

Configure your Gemini API key by running the command:
`CodeCompass: Set Gemini API Key`

Enjoy!
